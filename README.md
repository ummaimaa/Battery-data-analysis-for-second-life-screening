# NASA Battery Dataset — Second-Life SOH Screening

A MATLAB-based time-frequency analysis pipeline for estimating **State of Health (SOH)** of lithium-ion batteries and classifying them for second-life applications, using the NASA Prognostics Center of Excellence Battery Dataset.

---

## Overview

This project applies **Short-Time Fourier Transform (STFT)** analysis to discharge voltage signals to extract spectral features that correlate with battery degradation — enabling fast SOH estimation without requiring full charge/discharge capacity tests.

Batteries are classified into three health categories:

| Class | SOH Range | Recommendation |
|---|---|---|
| **Good** | ≥ 80% | Continue primary use |
| **Second-Life** | 60 – 79% | Repurpose for stationary storage, backup power, or low-speed EVs |
| **Degraded** | < 60% | Retire and recycle |

## Repository Structure

```
├── BatterySignalAnalysis.m       # Stage 1: Data download, preprocessing, STFT analysis
├── RuleBased_SOH_Screening.m     # Stage 2: Feature extraction, SOH estimation, classification
├── BatteryPlots/                 # Raw signal and STFT spectrogram plots (auto-generated)
├── FeatureResults/               # SOH distribution, classification, and feature plots (auto-generated)
├── allBatteries.mat              # Saved battery structs (generated by Stage 1)
└── feature_table.mat             # Final feature + classification table (generated by Stage 2)
```

---

## Pipeline

```
NASA Dataset (.zip)
      │
      ▼
BatterySignalAnalysis.m
  ├── Download & extract dataset automatically
  ├── Randomly select 4 batteries (rng(42) for reproducibility)
  ├── Classify cycles: charge / discharge / impedance
  ├── Resample voltage signals to 1 Hz (PCHIP interpolation)
  ├── Dynamic segment selection (max current-derivative onset)
  ├── STFT spectrogram (Hann window, 64-sample, 50% overlap, 128-pt FFT)
  └── Save allBatteries.mat → BatteryPlots/
      │
      ▼
RuleBased_SOH_Screening.m
  ├── Sliding-window feature extraction (120-sample segments, 60-sample step)
  ├── Spectral features: Band Energy (0.05–0.2 Hz), Spectral Centroid, Spectral Entropy
  ├── SOH estimation: capacity-based (primary) or band-energy proxy (fallback)
  ├── Rule-based classification: Good / Second-Life / Degraded
  ├── Confidence scoring (distance from nearest boundary)
  └── Save feature_table.mat → FeatureResults/
```

## Requirements

- **MATLAB R2020a or later**
- Signal Processing Toolbox (for `spectrogram`)
- Internet connection (for automated dataset download on first run)

No additional toolboxes or external dependencies required.

## Usage

Run the two scripts in order:

```matlab
% Stage 1 — Signal analysis and preprocessing
run('BatterySignalAnalysis.m')

% Stage 2 — SOH estimation and classification
run('RuleBased_SOH_Screening.m')
```

All outputs (plots and `.mat` files) are saved automatically. On first run, the NASA dataset (~50 MB) is downloaded and extracted automatically via `websave` and `unzip`.

## Features Extracted

| Feature | Frequency Band | Description |
|---|---|---|
| **Band Energy** | 0.05 – 0.2 Hz | Mean spectral power in the low-frequency diffusion band |
| **Spectral Centroid** | Full spectrum | Weighted centre of gravity of the power spectrum |
| **Spectral Entropy** | Full spectrum | Shannon entropy of the normalised power distribution |

These features were selected based on their documented correlation with electrochemical degradation mechanisms — specifically SEI layer growth (increasing ohmic resistance) and active material loss (increasing diffusion resistance), both of which shift energy distribution in the low-frequency voltage response.

## Results

Analysis of 4 randomly selected batteries (B0031, B0054, B0045, B0039) across **14,250 total segments**:

| Battery | Avg SOH | Classification |
|---|---|---|
| B0031 | 87.2% | **Good** |
| B0039 | 82.2% | **Good** |
| B0054 | 47.6% | **Degraded** |
| B0045 | 35.3% | **Degraded** |

Overall classification breakdown: 29.6% Good · 0.3% Second-Life · 70.0% Degraded

## Dataset

**NASA Prognostics Center of Excellence Battery Dataset**  
Saha, B. and Goebel, K. (2007). *Battery Data Set*. NASA Ames Prognostics Data Repository.  
https://ti.arc.nasa.gov/tech/dash/groups/pcoe/prognostic-data-repository

The dataset contains cycling data for 18650 lithium-ion cells run to end-of-life (30% capacity fade from nominal 2.0 Ah) under constant-current discharge profiles.

---

## Reproducibility

Battery selection uses `rng(42)` — running the scripts multiple times will always select the same 4 batteries and produce identical results. All intermediate data is saved to `.mat` files to allow Stage 2 to be re-run independently without re-downloading or re-processing Stage 1.

---

## Limitations

- Analysis is limited to the discharge voltage signal; temperature and current features are not used for SOH estimation
- The cross-battery band energy–capacity correlation is weak (r = −0.044) due to differing ageing histories; within-battery correlation is more meaningful
- Classification thresholds (80% / 60%) follow industry standards and may need adjustment for specific applications
- The NASA dataset covers only small-format 18650 cells under controlled laboratory conditions
